My Journey

Reflections on growth, challenges, and learning in the world of development.

What I Learned

Building this portfolio taught me the importance of clean code structure and accessible design. I mastered modern CSS features like CSS variables for theming, CSS Grid/Flexbox, and subtle animations that enhance user experience. I also gained valuable experience with Git version control, learning to create meaningful commits, manage branches, and maintain a professional project history. The process reinforced the value of planning before coding and iterating based on feedback from the developer community.

Challenges Faced

The biggest challenge was implementing a smooth theme toggle that transitions seamlessly between light and dark modes. I also struggled with creating a responsive navigation bar that works on all devices. Through persistence and research, I learned to use media queries effectively and discovered the importance of testing across different screen sizes throughout development. Version control with Git presented its own learning curve, but mastering commit conventions and branch management proved invaluable.
